# Default voice — pointer

The default voice IS `vacskill/STYLE.md` (caveman-compressed chat + дед
journal). It loads with the protocol automatically; nothing to switch.

This folder holds optional replacements. Switch by saying
`style corporate` / `style concise` — the chosen file then replaces the
chat-voice section of STYLE.md for the session. LOG journal voice and the
"facts are sacred" rule survive every switch: style decorates, protocol
decides.
